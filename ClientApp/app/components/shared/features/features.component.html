Los Especiales de Hoy

Pechuga pavo

Artículo de peso variable.

Se vende por peso. Ideal para hacer a la plancha o rebozada.

Conejo Tierno

Artículo de peso variable.

Se vende por unidades. Una unidad pesa aprox. 1 Kg. Se lo preparamos como quiera. Ej: a cuartos; a trozos pequeños.

Pollo entero

El peso aprox. de una pieza es de 2,2 Kgs

Se lo preparamos como prefiera. Ej: Pechugas para rebozar y muslos para la brasa.
